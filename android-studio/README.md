## About

[Android Studio](https://google.net) packaged for Fedora and published to [copr](https://copr.fedorainfracloud.org/coprs/myriad-sun/android-studio-git)

This package is updated every Sunday to track the latest upstream Lutris changes.

### Bugs

- Bugs related to Android Studio application should be reported to the [Android Studio GitHub org](https://github.com/lutris/lutris/issues)
- Bugs related to this package should be reported to [this Git project](https://github.com/lost-saint/copr/issues)

## Installation

1. Enable copr repo

```bash
sudo dnf copr enable myriad-sun/android-studio
```

- Substitute `dnf` for `yum` if desired

2. Install

```bash
sudo dnf install android-studio --refresh
```
