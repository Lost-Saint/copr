## About

### Bugs

- Bugs related to Lutris application should be reported to the [lutris GitHub org](https://github.com/lutris/lutris/issues)
- Bugs related to this package should be reported to [this Git project](https://github.com/lost-saint/copr/issues)

## Installation

0. Enable copr repo

```bash
sudo dnf copr enable myriad-sun/lutris
```

- Substitute `dnf` for `yum` if desired

0. Install

```bash
sudo dnf install lutris --refresh
```
