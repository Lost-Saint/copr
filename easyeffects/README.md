## About

### Bugs

- Bugs related to Easy Effects application should be reported to the [wwmm GitHub org](https://github.com/wwmm/easyeffects/issues)
- Bugs related to this package should be reported to [this Git project](https://github.com/lost-saint/copr/issues)

## Installation

1. Enable copr repo

```bash
sudo dnf copr enable myriad-sun/easyeffects
```

- Substitute `dnf` for `yum` if desired

2. Install

```bash
sudo dnf install easyeffects --refresh
```
